002 : Discord bot

Below is the discord bot I built. At first, I did basic tasks the bot could do. For example assigning roles and messaging in the chat. Then I tried kick and banning people from a custom server as a test. After I was more comfortable with the API and the python module for discord, I tried to be a bit more adventerous and make a music bot. I had some trouble at the start as I was able to download the music videos using the youtube dl module, but I was not able to play it and hear it inside the voice chat. So I went to youtube to help me as guidance to continue this project. After watching youtube, I went to patch my code up and insert the missing functions. Fortunatly they worked!
